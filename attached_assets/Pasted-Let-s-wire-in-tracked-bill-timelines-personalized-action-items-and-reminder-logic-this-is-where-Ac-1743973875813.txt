2. Action Item Generator (AIExplain Add-on)

Logic:
If bill is:
	•	In Committee → Show: “Contact committee members”
	•	Hearing scheduled → Show: “Testify or submit written comment”
	•	Up for vote → Show: “Call your rep before [date]”

Example Output:

"This bill is in the Senate Committee on Education. You can submit testimony or call key members today."

Optional Smart Template System:

function generateAction(status, chamber) {
  if (status.includes("Hearing")) return "Submit public comment or prepare to testify.";
  if (status.includes("In Committee")) return `Call committee members in the ${chamber}.`;
  if (status.includes("Scheduled for Vote")) return "Contact your rep before it’s too late.";
  return "Track updates and share this bill.";
}
