Step-by-Step Deployment Plan

Step 1: Upload to GitHub
	1.	Go to https://github.com/new
	2.	Create a repo named: act-up
	3.	On your machine:

unzip act-up-starter.zip
cd act-up
git init
git remote add origin https://github.com/yourusername/act-up.git
git add .
git commit -m "Initial commit with starter files"
git push -u origin main



⸻

Step 2: Setup Supabase
	1.	Go to https://supabase.com
	2.	Create a free project.
	3.	Set up a Postgres database (name it actup)
	4.	Go to the SQL Editor and paste any schema you want (I can help generate one).
	5.	Copy the Project URL and Anon Key from Project Settings → API.

Update your .env with:

DATABASE_URL=supabase-postgres-connection-string
SUPABASE_ANON_KEY=your_anon_key



⸻

Step 3: Connect Frontend (Optional Now, Easy Later)

When ready:
	1.	Deploy your React frontend to Vercel

	3.	Vercel will auto-deploy every time you push changes

Bonus: Add AI Bill Explainer (Beta Tool)

When you want to integrate GPT for personalized bill summaries:
	1.	Get your OpenAI API key: https://platform.openai.com/account/api-keys
	2.	Use something like:

import openai

openai.api_key = os.getenv('OPENAI_API_KEY')

def explain_bill(text):
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[{"role": "user", "content": f"What does this bill mean for a Texas teacher: {text}"}]
    )
    return response.choices[0].message['content']
