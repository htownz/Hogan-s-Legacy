🚀 Deploy Act Up Backend on AWS

Option A: AWS EC2 (Straightforward FastAPI Deployment)

If you’re comfortable with SSH and instance management:

🔧 1. Launch EC2 instance
	•	Ubuntu 22.04 or Amazon Linux 2023
	•	Open ports: 22, 80, 443, and 8000 (for dev)
	•	Assign an Elastic IP (optional but helpful)

🧩 2. Connect & Install Dependencies

ssh -i "your-key.pem" ec2-user@your-ec2-ip

# Update & install
sudo apt update && sudo apt upgrade -y
sudo apt install python3-pip python3-venv nginx git -y

📂 3. Upload your code (from local machine)

Either scp or clone from GitHub:

git clone https://github.com/your-username/act-up.git
cd act-up
python3 -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt

🚀 4. Run FastAPI (Dev mode)

cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

Test via: http://your-ec2-ip:8000
